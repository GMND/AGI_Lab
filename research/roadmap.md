# Roadmap: Project AGI-Lab

## Phase 1: Theoretical Foundation (Current)
- [ ] Формализация математической модели (World Model + LAM).
- [ ] Обзор литературы (SOTA в области World Models и Hierarchical RL).
- [ ] Определение метрик успеха (Generalization, Sample Efficiency, Reasoning).

## Phase 2: Simulation Environment & Architecture Design
- [ ] Выбор/создание среды (например, Brax или специализированная среда для манипуляций).
- [ ] Разработка прототипа латентного пространства (VAE/VQ-VAE/JEPA).
- [ ] Реализация базового планировщика (MCTS или Diffusion Policy).

## Phase 3: Implementation & Scaling
- [ ] Обучение базовой модели (Small Scale).
- [ ] Масштабирование архитектуры.
- [ ] Тестирование на задачах с долгосрочным планированием.

## Phase 4: Evaluation & Publication
- [ ] Бенчмаркинг против существующих SOTA.
- [ ] Написание статьи для NeurIPS/ICLR.
