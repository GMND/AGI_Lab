# AGI Lab

Исследовательский репозиторий по поиску и реализации архитектур общего искусственного интеллекта (AGI).

## Текущий фокус
- Исследование Large Action Models (LAM) и их способности к автономному взаимодействию с внешними средами.
- Разработка методов токенизации действий (Action Tokenization).
- Создание среды для тестирования closed-loop взаимодействия (Action-Feedback Loop).

## Структура исследования
- `RESEARCH/LAM_Study/`: Теоретическая база, гипотезы и планы реализации.
  - `ACTION_TOKENIZATION.md`: Теоретическая модель.
  - `IMPLEMENTATION_PLAN.md`: Архитектура системы и дизайн классов.
  - `HYPOTHESIS.md`: Проверяемые утверждения.
- `src/`: Реализация ядра LAM.
  - `hierarchical_tokenizer.py`: Иерархический токенизатор.
  - `hierarchical_env.py`: Обертка Gymnasium.
- `scripts/`: Скрипты для верификации и запуска экспериментов.
  - `verify_run.py`: Проверка корректности токенизатора.
- `EXPERIMENTS/`: Логи и данные экспериментов.
