# AGI Lab

Исследовательский репозиторий по поиску и реализации архитектур общего искусственного интеллекта (AGI).

## Текущий фокус
- Исследование Large Action Models (LAM) и их способности к автономному взаимодействию с внешними средами.
- Разработка методов токенизации действий (Action Tokenization).
- Создание среды для тестирования closed-loop взаимодействия (Action-Feedback Loop).

## Структура исследования
- `RESEARCH/LAM_Study/`: Теоретическая база и гипотезы.
- `EXPERIMENTS/`: (In development) Логи и данные экспериментов.
