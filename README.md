# AGI Lab (GMND)

Добро пожаловать в исследовательскую лабораторию AGI. Наша цель — разработка и изучение архитектур, ведущих к сильному искусственному интеллекту (AGI).

## Текущий фокус
- **Large Action Models (LAM):** Исследование моделей, способных к планированию и выполнению действий в сложных средах.

## Структура проекта
- `/research`: Теоретические обоснования, обзоры литературы и дорожные карты.
- `/experiments`: Код и логи экспериментов.
- `/docs`: Техническая документация.
- `/src`: Базовые библиотеки и архитектуры.
